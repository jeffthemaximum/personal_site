Create a Python List

You just need to provide name of the list and initialize it with values.Following is an example of a List in Python:


>>> myList = ["The", "earth", "revolves", "around", "sun"]
>>> myList
["The", "earth", "revolves", "around", "sun"]

Accessing items in a List

Lists in python are zero indexed. This means, you can access individual elements in a list just as you would do in an array. Here is an example:


>>> myList[0]
'The'
>>> myList[4]
'sun'

Lists cannot be accessed beyond the rightmost index boundary. Here is an example:


>>> myList[5]
Traceback (most recent call last):
  File "", line 1, in 
IndexError: list index out of range

If you try to access value at a negative index then values from the rightmost index can be accessed backwards. Here is an example:


>>> myList[-1]
'sun'

The above sections, "Create a Python List", and "Accessing items in a list" borrow heavily from http://www.thegeekstuff.com/2013/06/python-list/.

List methods

Below is a list of some of the methods that Python includes for lists

list.append(obj)

The method append() appends a passed obj into the existing list. Here is an example:


>>> myList.append("today")
["The", "earth", "revolves", "around", "sun", "today"]
>>> myList.append(["because", "of", "gravity"])
["The", "earth", "revolves", "around", "sun", "today", ["because", "of", "gravity"]]
list.extend(seq)

The method extend() appends the contents of seq to list. Here is an example:


>>> myList = ["The", "earth", "revolves", "around", "sun"]
>>> myList.extend(["today", "because", "of", "gravity"])
["The", "earth", "revolves", "around", "sun", "today", "because", "of", "gravity"]

*Bonus thought question #1: Why did I have to reinitialize myList to equalling ["The", "earth", "revolves", "around", "sun"] ?

list.insert(index, obj)

The method insert() inserts object obj into list at offset index.


>>> myList = ["The", "earth", "revolves", "around", "sun"]
>>> myList.insert(4, myList)
["The", "earth", "revolves", "around", "the", "sun"]
list.reverse()

The method reverse() reverses objects of list in place.


>>> myList = ["The", "earth", "revolves", "around", "sun"]
>>> myList.reverse()
["sun", "around", "revolves", "earth", "The"]
