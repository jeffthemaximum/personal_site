Create a Ruby Array

You just need to provide the name of the Array and initialize it with values. Following is an example of initializing an Array with Ruby:


> my_array = ["The", "earth", "revolves", "around", "sun"]
=> ["The", "earth", "revolves", "around", "sun"]

Accessing elements in an Array

Arrays in Ruby are zero indexed. This means, you can access individual elements in an array by the order the appear in the Array. Here is an example:


> my_array[0]
=> 'The'
> my_array[4]
=> 'sun'

Arrays cannot be accessed beyond the rightmost index boundary. Ruby handles this fairly 'gracefully' by returning nil, where other languages, such as Python, might return an error. Here is an example:


> my_array[15]
=> nil

If you try to access values at a negative index then values from the rightmost index can be accessed backwards. Here is an example:


> my_array[-1]
=> 'sun'

The above sections, "Create a Ruby Array", and "Accessing items in an Array" borrow heavily from http://www.thegeekstuff.com/2013/06/python-list/ and https://ruby-doc.org/core-2.4.0/Array.html.


Array methods

Below is a list of some of the methods that Ruby includes for arrays

array.push(obj, ...)

The method push() pushes (or appends) passed object(s) into the existing array. Here is an example:


> my_array.push("today")
=> ["The", "earth", "revolves", "around", "sun", "today"]
> my_array.push("today")
=> ["The", "earth", "revolves", "around", "sun", "today", ["because", "of", "gravity"]]
array.concat(other_array1, ...)

The method concat() appends the contents of an other array to the original array. Here is an example:


> my_array = ["The", "earth", "revolves", "around", "sun"]
> my_array.concat(["today", "because", "of", "gravity"])
=> ["The", "earth", "revolves", "around", "sun", "today", "because", "of", "gravity"]

*Bonus thought question #1: Why did I reinitialize my_array to equalling ["The", "earth", "revolves", "around", "sun"] ?

array.insert(index, obj...)

The method insert() inserts the given values before the index position.


> my_array = ["The", "earth", "revolves", "around", "sun"]
> my_array.insert(4, "the")
=> ["The", "earth", "revolves", "around", "the", "sun"]
> my_array.insert(1, "big", "blue")
=> ["The", "big", "blue", "earth", "revolves", "around", "the", "sun"]
array.reverse()

The method reverse() reverses objects of list in place.


> my_array = ["The", "earth", "revolves", "around", "sun"]
=> ["sun", "around", "revolves", "earth", "The"]
